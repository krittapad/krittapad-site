# Krittapad Portfolio Site

Static portfolio website ready for GitHub Pages.

## Edit these first

- `index.html` - name, hero text, featured work
- `showreels/` - project pages and reel links
- `resume.html` - resume summary and PDF link
- `contact.html` - email and profile links

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload every file in this folder.
3. Open repository Settings, then Pages.
4. Choose Deploy from branch, branch `main`, folder `/root`.
5. Wait for GitHub to publish the site URL.
